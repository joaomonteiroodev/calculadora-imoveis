# Calculadora de Financiamento Imobiliário

Calculadora desenvolvida em Python que calcula o financiamento de imóveis pelo sistema **Tabela Price**.

## Funcionalidades

- Calcula a parcela mensal fixa
- Mostra o total pago ao longo do financiamento
- Mostra o total pago em juros
- Valida os dados inseridos pelo usuário
- Permite calcular várias vezes sem reiniciar

## Como usar

1. Execute o programa no terminal:
   python calculadora.py

2. Informe os dados solicitados:
   - Valor do imóvel
   - Valor da entrada
   - Taxa de juros mensal (%)
   - Prazo em meses

3. O programa exibe o resultado e pergunta se deseja calcular novamente.
